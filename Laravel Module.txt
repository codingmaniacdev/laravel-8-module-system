Laravel Module
----------------------------------------
1. php artisan module:make UserManagement - create Module.
2. php artisan module:make-model User UserManagement - create model in module.
3. php artisan module:make-factory User UserManagement - create factory.
4. php artisan module:make-seed User UserManagement - create seed.
5. php artisan module:seed UserManagement --class=UserTableSeeder - create seed.
6. php artisan module:make-migration create_users_table UserManagement - create migration.
7. php artisan module:migrate UserManagement - migrate specific Module migration.
